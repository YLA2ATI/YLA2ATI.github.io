How to Connect

🖥️ PC Setup

1

⚡ Auto Setup Recommended

  1. Download the setup file: GT_hosts.bat
  2. Run as Administrator — right-click GT_hosts.bat → Run as administrator. It sets everything up automatically.
  3. Relaunch — log out and fully close Growtopia.
  4. Reopen and log back in — you're in! 🎉
2

🔧 Manual Setup

Use this if the auto setup didn't work.

  1. Go to C:\Windows\System32\drivers\etc
  2. Open the hosts file as Administrator
  3. Add these two lines at the very end: 
    45.150.149.37 www.growtopia1.com
45.150.149.37 www.growtopia2.com
  4. Save → log out of Growtopia → reopen & log in.

📱 Android Setup

1

⚡ Modded APK Recommended

  1. Uninstall your current Growtopia app.
  2. Download & install the PowerKuy client: Growtopia.apk. Allow installs from unknown sources if prompted.
  3. Open the app, log in, and enjoy! ▶️
2

🔧 PowerTunnel

Use this if the modded APK didn't work.

  1. Download PowerTunnel.
  2. Open PowerTunnel and tap the square icon (top right).
  3. Scroll to Hosts and tap the settings icon next to it.
  4. Tap Hosts File URL, paste https://powerkuy.com/hosts.txt, then tap OK.
  5. Log out of Growtopia → log back in — you're in!

🍎 iOS Setup

1

📲 Surge 5

  1. Download Surge 5 from the App Store.
  2. Launch Surge 5 and tap Default.conf.
  3. Go to Import → Download Profile from URL, paste https://powerkuy.com/ios.config, then tap OK.
  4. Tap Setup and allow the VPN configuration when prompted.
  5. Open Growtopia → log out of your GrowID → log back in — you're in!
